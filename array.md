## Array Interview question
1. Way to create Array
  There are 2 way to create Array
  - ```new Array()```
      
      let a = new Array();
      a['name'] = 'Ashok';
      console.log(a); // ['name': 'Asok']
   - ```[]```

      let a = [];
      a['name'] = 'Ashok';
      console.log(a); // ['name': 'Asok']
